
### ONE一句

#### 普通请求

请求类型：GET   

[API测试平台](https://api.xygeng.cn/test.html)

接口地址：

```
https://api.xygeng.cn/one
```

接口结果：

```json
{
    "code": 200,
    "data": {
        "id": "807",
        "tag": "漫画",//主题
        "origin": "《萤火之森》",//来自
        "content": "其实美丽的故事都是没有结局的，只因为它没有结局所以才会美丽。",//内容
        "datetime": "1548230343"
    }
}
```



####  直接调用

直接在网页上插入下面代码即可

```html
<script type="text/javascript" src="https://api.xygeng.cn/one/js/"></script>
```
